# whatsapp_analyzer
Analyzes chat history from whatsapp for most used words, amounts per month and other stuff
**Now supports group chats!**
## How to use
- Send your whatsapp chat history via E-Mail and save it in this folder as "chat.txt"
- run analyzer.py
